前言

必读#

  • 在生产环境发布前前，请先通过测试环境调试完所有接口及流程后待我方确认无误方可上生产环境

  • 在开发过程中，可以使用接口调试工具来在线调试某些接口。

  • 在开发出现问题时，可以通过接口调用的返回码查看失败原因或联系我方技术支持人员

  • 所有接口都采用POST方式（包括查询接口）

  • 请求和响应的数据都为JSON格式。注：请设置Header的Content-Type为application/json，采用UTF-8字符编码

  • 所有的接口必须进行接口签名
