## Install

### Install from `npm`:

```bash
$ npm install artplayer-plugin-control
```

### Install from `yarn`:

```bash
$ yarn add artplayer-plugin-control
```

### Install from `script`:

```html
<!-- local -->
<script src="path/to/artplayer-plugin-control.js"></script>

<!-- jsdelivr -->
<script src="https://cdn.jsdelivr.net/npm/artplayer-plugin-control/dist/artplayer-plugin-control.js"></script>

<!-- unpkg -->
<script src="https://unpkg.com/artplayer-plugin-control/dist/artplayer-plugin-control.js"></script>
```

Then you can access the plugin function through `window.artplayerPluginControl`
